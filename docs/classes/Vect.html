    Vect

    Represents a vector as well as a point.

    All Methods on a Vector that return a vector are chainable and will change the initial vector to the result of the operation.

    rotateTowards
    (
    • angle
    • stepSize
    )
    Vect
    chainable

    Rotates the vector towards the given angle. The rotation is limited by stepSize, so move a 0 vector towards 90 with limit 50 would result in a 50 vector. If the method with the same limit is reapplied again the result would be 90.

    The rotation will take the "shortest way" towards the given angle.

    A 180 rotation will be clockwise(TODO???)

    Parameters:

    • angle Number
      • angle in radians
    • stepSize Number
      • maximum angle to move towards angle

    Returns:

    Vect:

    the rotated vector
